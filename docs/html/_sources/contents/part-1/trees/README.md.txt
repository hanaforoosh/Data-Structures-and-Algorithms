# درخت‌‌ها

درخت گرافی همبند و بدون دور است. در شکل زیر یک نمونه درخت نمایش داده شده است. درخت‌ها ساختمان داده‌ی مهمی در علم کامپیوتر هستند. شاید فکر کنید که استفاده از چنین ساختمان داده‌ای چه فایده‌ای دارد؟ در ادامه‌ی فصل به انواع مختلف و کاربرد آن‌ها می‌پردازیم که مطالب این بخش‌ها می‌تواند پاسخی برای این سؤال پاسخ باشد.
