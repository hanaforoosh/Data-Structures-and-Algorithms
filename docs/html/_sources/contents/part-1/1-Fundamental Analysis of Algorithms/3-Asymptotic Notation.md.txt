# نمادهای مجانبی

برای مقایسه‌ی رشد توابع برای اینکه بیان کنیم دو تابع نسبت به یکدیگر چطور رفتار می‌کنند از نماد‌های مجانبی استفاده می‌شود که به شرح زیر است:

## O
مجموعه توابعی که **کران بالا** برای تابع تعیین می‌کنند.  به عبارت دیگر توابعی که نمودار آن‌ها از یک نقطه‌ای به بعد بالاتر از نمودار g(n) قرار می‌گیرد. برای مثال در هر سه نموداری که در شکل زیر وجود دارد تابع f(n) بالاتر از تابع g(n) قرار گرفته است.  

![مقایسه‌ی رشد توابع](/_static/O.png)

  این مجموعه توابع به صورت ریاضی به شکل زیر تعریف می‌شوند:

$$
O(g(n))={f(n):\exists C,n_0>0,\forall n \ge n_0: 0 \le f(n) \le c g(n)}
$$

## Ω
مجموعه توابعی که **کران پایین** برای تابع تعیین می‌کنند. به عبارت دیگر توابعی که نمودار آن‌ها از یک نقطه‌ای به بعد پایین‌تر از نمودار g(n) قرار می‌گیرد. برای مثال در هر سه نموداری که در شکل قبل وجود دارد تابع f(n) پایین‌تر از تابع g(n) قرار گرفته است.  


  این مجموعه توابع به صورت ریاضی به شکل زیر تعریف می‌شوند:
  
$$
Ω(g(n))={f(n):\exists C,n_0>0,\forall n \ge n_0: 0 \le c g(n) \le f(n)}
$$


``` important:: همانطور که احتمالا از توضیحات دو نماد مجانبی O و Ω متوجه شده‌اید، این دو نماد برعکس یکدیگرند، به عبارت دیگر اگر f(n)=O(g(n))  آنگاه g(n)=Ω(f(n))  
```

## θ
مجموعه توابعی که **هم کران بالا و هم کران پایین** برای تابع تعیین می‌کنند به عبارت دیگر اگر بتوان دو عدد صحیح (C1 و C2) پیدا کرد به طوری که با ضرب آن‌ها در تابع g(n) نمودار تابع f(n) از یک نقطه‌ای به بعد بین دو تابع C1g(n) و C2g(n) قرار گیرد. برای مثال در نموداری که در شکل زیر وجود دارد، با انتخاب مقادیر ۲ و ۱/۲ برای C1 و C2، تابع f(n) بین دو تابع قرار گرفته است 

![مقایسه‌ی رشد توابع](/_static/teta.png)

این مجموعه توابع به صورت ریاضی به شکل زیر تعریف می‌شوند:
$$
θ(g(n))={f(n):\exists C1,C2,n_0>0,\forall n \ge n_0: 0 \le C1 g(n) \le f(n) \le C2 g(n)}
$$

## o
مجموعه توابعی که کران بالا برای تابع تعیین می‌کنند اما هیچگاه برابر با اندازه تابع نمی‌شود. تفاوت این نماد با «نماد O» این است که برای استفاده از «نماد O» تنها لازم بود یک ضریب پیدا کنیم که f(n) < c g(n) باشد، امّا برای استفاده از «نماد o» باید به ازای  **همه‌ی ضریب‌های ممکن** باید f(n) < c g(n) باشد.
این مجموعه توابع به صورت ریاضی به شکل زیر تعریف می‌شوند:
$$
o(g(n))={f(n):\exists C,n_0>0,\forall n \ge n_0: 0 \le f(n) < c g(n)}
$$

## ω
مجموعه توابعی که کران بالا برای تابع تعیین می‌کنند اما هیچگاه برابر با اندازه تابع نمی‌شود. تفاوت این نماد با «نماد Ω» این است که برای استفاده از «نماد O» تنها لازم بود یک ضریب پیدا کنیم که f(n) > cg(n) باشد، امّا برای استفاده از «نماد ω» باید به ازای  **همه‌ی ضریب‌های ممکن** باید f(n) > cg(n) باشد.
$$
ω(g(n))={f(n):\exists C,n_0>0,\forall n \ge n_0: 0 \le c g(n) < f(n)}
$$
